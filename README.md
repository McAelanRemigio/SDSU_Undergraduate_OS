## Overview

This repository is a curated collection of my undergraduate coursework and applied academic work in **Data Science** and **Leadership** at **San Diego State University (SDSU)**.

It serves two main purposes:

- As a **long-term reference** for concepts and skills I learned throughout undergrad (because it’s surprisingly easy to forget things after four years)
- As an **academic and professional portfolio** demonstrating that I can meaningfully explain, apply, and reflect on the tools and ideas I studied

This repository intentionally does **not** function as a homework archive. I do not upload graded exams or assignments. Instead, it focuses on work that reflects **genuine learning, application, or synthesis**.
